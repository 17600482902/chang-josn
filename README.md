## 功能

将数组转换成josn
输入   [{a:1},{b:6}];
返回    {a:1,b:6}
## 安装下载
-下载地址 https://github.com/17600482902/chang-josn/tree/master
- npm i @zhoudiankai/change-json

## 快速使用
    import { changeJson } from "@zhoudiankai/change-json";
    let c = changeJson([ {a: 1, b: 2} ]);
	console.log(c);
